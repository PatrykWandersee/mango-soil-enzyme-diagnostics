# Mango Soil Enzyme Diagnostics

This repository documents a reproducible analytical workflow for evaluating soil enzyme activity and biological soil-quality indicators in irrigated Palmer mango orchards in the Brazilian semiarid region.

The project is intended to support a manuscript focused on enzymatic indicators, biological diagnostic interpretation, and their relationship with soil properties and relative orchard productivity.

## Data availability

The original dataset is private and is not included in this repository. Public examples will use synthetic data with the same structure as the private analytical dataset, generated only for workflow demonstration.

## Planned workflow

- organize biological, chemical, physical, and productivity variables;
- evaluate beta-glucosidase, arylsulfatase, and GMea;
- explore relationships with clay, soil organic matter, salinity/sodicity, and relative yield;
- compare raw and normalized enzyme indicators;
- test diagnostic approaches for biological interpretation;
- generate manuscript-oriented tables and figures.
